JS数据类型转换

转化为 string 类型

① 和空字符’’串相加 ， 1 + ‘’ = “1” ， true + ‘’ = “true”
② 使用 .tostring() 方法 ， (11).toString() // “11”
③ 使用 String() 方法 ， window.String(‘111’) // “111” ，String(true)//“true”

转化为 boolean 类型

① 使用 Boolean() 方法 ， Boolean(1) // true ， Boolean(null) //false
② 取反两次得到本身对应布尔值 ， !!true // true， !!{} // true ,对象布尔值全都是true
五个falsy值
number：0， NaN
string：’’（空字符串）
null： null
undefined： undefined

转化为 number 类型

① Number(‘111’) // 111
② parseInt(‘1’,10) // 1,10代表十进制
③ parseFloat(‘1.23’) // 1.23
④ 和零相减：’11’ - 0 // 11
⑤ + 取原有值: + ‘12’ //12
区分点：
parseInt(‘011’) // 11
parseInt(‘011’ , 8) // 9
parseInt(‘s’) // NaN
parseInt(‘1s’) // 1

内存泄漏： 由于浏览器垃圾处理机制的原因，使得该被标记为垃圾的没被标记，就会造成内存泄漏

浅拷贝 VS 深拷贝

深拷贝：b变不影响a，基本类型赋值就是深拷贝，内存里的值重新复制一份改写，不影响别的
浅拷贝：b变影响a，仅仅拷贝了stack地址，内存那块并没有拷贝，改变会造成影响
