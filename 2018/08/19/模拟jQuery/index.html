模拟jQuery

window.jQuery

window.jQuery = ???
window.$ = jQuery

var $div = $(‘div’)
$div.addClass(‘red’) // 可将所有 div 的 class 添加一个 red
$div.setText(‘hi’) // 可将所有 div 的 textContent 变为 hi

实现过程

通过自己封装一个 window.jQuery 这样的练习，可以更清晰地对 jQuery 的本质有初步的认知。

1、从封装一个函数入手，对于一段可以反复调用的代码块，通常将其包装成一个函数方便反复调用，本例中封装了两个具有特殊意义的函数 

addClass()     //将所有 div 的 class 添加一个 red         
settext()   //将所有 div 的 textContent 变为 hi

2、这些特殊函数被装入一个新的接口对外提供使用，也就是本例中window.jQuery 这个函数

3、window.jQuery 首先判断输入是一个节点还是选择器，组装成相应的对象作为全局变量，为后面所有操作提供对象，所以 nodes 和下文各个函数组合都称为闭包。方法调用时相当于隐形的调用call方法将当前对象作为第一个参数传入。

4、addClass()实现：遍历输入的class数组，给相应的元素赋值；
settext()实现：找到相应节点给元素赋值；
可以在里面添加任何想要实现的方法供外部使用，
如完善text() 方法同时具备取值和赋值功能，
