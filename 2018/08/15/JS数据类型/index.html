JS数据类型

JavaScript共7中数据类型（ES6引入了Symbol类型）

number
十进制：1 ， 1.1 ， .1 ， 1.23e2
二进制（0b/0B开头）：0b11， 0B12
八进制（0开头）：011
十六进制（0x/0X开头）：0x11， 0X11

String
‘你好’， “你好”，’’， “”， ‘ ‘，” “，空字符串长度为0，空格字符串长度为1
转义字符串（反斜杠\）:
‘\’’ 代表’ ， ‘\\’代表’\’ ， ‘\n’ 回车 ， ‘\t’ Tab 这些转义的键长度都是1
多行字符穿（字符串中有换行） 不推荐，可以使用字符相加var s = ‘123’ + ‘456’

Boolean
true false 常用语if判断，&&和||

null ， undefined 都表示什么也没有
null通常用于表示一个空对象，undefined通常用于表示一个空的非对象

Object（哈希表）
key只接受字符串 ,由基本类型组成 

var person = {
           name: 'frank',    
           age: 18,
           married: true, 
           xiaosan: undefined, 
           child:{name:'ming',age: 1}
           }

person[‘name’] ==> person.name; 当’ ‘中的内容符合标识符命名规范的情况下可以使用.取值
遍历 ： for(var key in person){ console.log(pserson[key]) }
